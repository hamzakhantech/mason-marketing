Tracking Dashboard

Enable and configure analytics providers for masononsite.com. All changes here generate an updated tracking.js file that you copy into the mason web folder. No deployment required — save the file and refresh the page to apply.

💡 Changes here don't auto-save to disk. Use the "Export tracking.js" page to copy the generated code, then paste it into tracking.js in the mason web folder.

Active trackers

Quick start

1. Click a tracker in the left nav
2. Toggle it on and paste your tracking ID
3. Click "Save to Dashboard"
4. Go to "Export tracking.js" and copy the generated code
5. Replace the contents of tracking.js in the mason web folder
6. Reload any page on the site to verify in DevTools → Network

Google Analytics 4

GA4 tracks pageviews, events, and user behaviour across masononsite.com. Your Measurement ID looks like G-XXXXXXXXXX. Find it in Google Analytics → Admin → Data streams → your stream → Measurement ID.

Google Analytics 4

Tracks pageviews, sessions, user demographics, traffic sources, and custom events. The most widely used web analytics platform. Required for Google Ads conversion tracking.

How to verify

Open DevTools (F12) → Network tab → reload the page → filter by "google-analytics.com".
You should see a request to gtag/js?id=G-XXXXXXX.
In GA4, go to Realtime → Overview. Your visit should appear within 30 seconds.

Plausible Analytics

Plausible is a privacy-first, cookie-free analytics tool. No GDPR consent banner required. Simpler than GA4 but still shows pageviews, bounce rate, traffic sources, and top pages. Good choice if you want clean data without the complexity of GA4.

Plausible Analytics

Privacy-first analytics. No cookies. No personal data. No consent banner needed. Lightweight script (~1 KB). Open source. Data stored in EU.

Microsoft Clarity

Clarity records heatmaps and session recordings — free, unlimited. Shows where users click, how far they scroll, and what they do before leaving. Find your Project ID in Clarity → Settings → Overview.

Microsoft Clarity

Free heatmaps and session recordings. Unlimited sessions. Integrates with GA4. Useful for understanding why visitors aren't converting on the pricing page.

Hotjar

Hotjar provides heatmaps, session recordings, and feedback polls. More polished than Clarity but paid after 35 sessions/day. Your Site ID is a number — find it in Hotjar → Settings → Tracking code.

Hotjar

Heatmaps, session recordings, feedback surveys, and funnel analysis. Free tier: 35 sessions/day. Paid from $32/month.

Meta Pixel

Required for Facebook and Instagram ads retargeting and conversion tracking. Your Pixel ID is a 15–16 digit number. Find it in Meta Business Suite → Events Manager → your pixel.

Meta Pixel

Tracks site visitors for Facebook/Instagram ad targeting. Required for retargeting campaigns, lookalike audiences, and conversion optimisation campaigns.

LinkedIn Insight Tag

Required for LinkedIn ads retargeting and conversion tracking. Your Partner ID is a 7-digit number. Find it in LinkedIn Campaign Manager → Account Assets → Insight Tag.

LinkedIn Insight Tag

LinkedIn website demographics (job titles, industries of your visitors) and conversion tracking for LinkedIn ad campaigns. Highly useful for B2B targeting.

Segment

Segment is a customer data platform — it collects events once and routes them to any destination (GA4, Mixpanel, Amplitude, Intercom, etc.) without adding more tracking scripts. Find your Write Key in Segment → Sources → your source → API Keys.

Segment

Unified customer data platform. Track once, send everywhere. Useful when you want to run multiple analytics tools without performance impact.
